# Registro de Decisiones de Diseño (ADR)

Este documento centraliza las decisiones arquitectónicas importantes tomadas durante el desarrollo del proyecto.

## Estructura de un ADR
1. **Título**: Nombre descriptivo de la decisión.
2. **Estatus**: Propuesto / Aceptado / Superado.
3. **Contexto**: ¿Por qué estamos tomando esta decisión ahora?
4. **Decisión**: La solución elegida.
5. **Consecuencias**: Impacto positivo y negativo.
